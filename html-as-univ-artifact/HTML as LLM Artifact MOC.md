# HTML as LLM Artifact MOC

Map of content for research on the idea that **HTML is the universal artifact for LLMs** — that a single self-contained HTML file is the ideal output format for AI-generated artifacts. Covers the thesis, the format alternatives, the interactive-UI standards, the tools, the security model, and the implications for Archetype. Current as of June 2026.
