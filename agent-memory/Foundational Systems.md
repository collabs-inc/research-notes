# Foundational Systems

The key systems (2023-2024) that established the core ideas in agent memory. Each has its own note with full detail.

## The Systems

- [[Generative Agents]] -- triple-weighted retrieval and reflection (Stanford, 2023)
- [[Reflexion]] -- natural language self-critique as [[Procedural Memory]] (2023)
- [[Voyager]] -- executable skill library as procedural memory (2023)
- [[Letta]] -- OS-inspired virtual memory management, originally MemGPT (2023)
- **ChatDB** -- databases as symbolic memory with SQL-like operations (2023)

## What They Proved

1. **Memory needs structure** -- raw storage without retrieval weighting and consolidation degrades quickly ([[Generative Agents]])
2. **Self-managed memory works** -- agents can effectively manage their own context ([[Letta]])
3. **Failure memory is powerful** -- post-mortems bootstrap learning without training ([[Reflexion]])
4. **Executable memory compounds** -- stored skills enable compositional capability growth ([[Voyager]])

These insights directly inform the [[Memory Middleware Players|middleware companies]] and [[Emerging Trends in Agent Memory|emerging trends]] of 2025-2026.
