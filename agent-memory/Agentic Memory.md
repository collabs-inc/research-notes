# Agentic Memory

The shift from passive, system-controlled memory storage to active, agent-directed memory management. Instead of hardcoded rules for what to remember and how to organize it, the agent itself decides.

## The Spectrum

Memory systems range from fully external to fully agent-controlled:

| Level | Who Decides | Example |
|-------|-------------|---------|
| **System-controlled** | External pipeline | [[Mem0]] (system extracts, stores, retrieves) |
| **Agent-triggered** | Agent calls memory functions | [[Letta]] (agent decides when to read/write) |
| **Agent-organized** | Memories self-organize | [[A-MEM]] (memories generate own descriptions and connections) |
| **Self-evolving** | Architecture itself adapts | [[MemEvolve]] (encode/store/retrieve modules reconfigure) |

The trend is moving rightward -- toward more agency in the memory system itself.

## Key Systems

**[[A-MEM]]** (NeurIPS 2025) -- memories generate their own descriptions, form connections, and evolve relationships. Inspired by the Zettelkasten method. The landmark paper defining this category.

**[[Letta]]** (MemGPT) -- the agent manages its own context via function calls. Pioneered the idea that agents should control their own memory operations.

**[[MemEvolve]]** (December 2025) -- goes further by evolving the memory architecture itself through meta-learning. Not just agentic content, but agentic structure.

**EvolveMem** (May 2026) -- applies AutoResearch to memory architecture evolution.

## Why It Matters

Fixed memory pipelines make assumptions about what's important, how to index it, and when to forget. These assumptions may not hold across:
- Different task domains
- Different user preferences
- Different information density levels

Agentic memory adapts to context rather than imposing a predetermined structure.

## Connection to Human Practices

The Zettelkasten inspiration in [[A-MEM]] is not coincidental. Human knowledge workers have long practiced active note-taking where notes link to each other and evolve over time (Obsidian, Roam Research, org-mode). Agentic memory brings this same self-organizing principle to AI systems.
