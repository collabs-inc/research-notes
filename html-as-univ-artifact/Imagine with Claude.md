# Imagine with Claude

Anthropic's temporary research preview (released ~29 Sept 2025, alongside Claude Sonnet 4.5) where Claude generates an entire software UI **in real time, with no prewritten code.** The purest "ephemeral generated software" experiment.

## What It Is

Anthropic's framing: *"Claude generates software on the fly. No functionality is predetermined; no code is prewritten."* It emulates a desktop environment (icons, a trash can, a "What do you want to build?" prompt) where the interface is synthesized per interaction by the model.

> The exact output mechanics (single HTML document vs. streamed components) were not stated in a primary source — treat the "HTML specifically" framing as inferred from the genre, not confirmed.

## Why It Matters

Imagine is the [[Generative UI|generative-UI]] bet taken to its limit: software as an *ephemeral stream*, regenerated each interaction rather than saved — the opposite end of [[The Artifact Landscape]] from [[Single-File HTML Apps|persistent single-file artifacts]]. It pairs with Google's [[Gemini Canvas and Generative UI|Dynamic View]] as the two frontier "no persistent artifact at all" experiments.
