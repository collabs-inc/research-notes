# v0

Vercel's generative-UI tool — prompt or screenshot → streamed **React + Tailwind + shadcn/ui** code. Represents the *framework-component* pole of [[The Artifact Landscape]], in contrast to the single-file-HTML pole.

- **Launched** October 2023 (public beta). Profiled in depth in the sibling vaults [[agentic-dev-envs/v0]] and [[product-engineering/v0]]; this note covers the artifact-format lens.

## How It Relates to HTML-as-Artifact

v0's output is **framework code** (React components, copy-paste into a repo), not a self-contained HTML file — it needs the build ecosystem to run. It illustrates the spectrum: where [[Claude Artifacts]] and [[tldraw Make Real]] produce a runnable HTML file, v0 produces production React *source*. Both ultimately render as HTML, but v0 optimizes for *shippable code* over *portable artifact* — see [[React and Generative UI]] ("React-as-source, HTML-as-artifact"). It originally generated only frontend components, not the backend needed to make them function.
