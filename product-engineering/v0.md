# v0

[[Vercel]]'s generative-UI tool — text/image → React using [[shadcn]]/ui + Tailwind, refined conversationally. Central to [[Guillermo Rauch]]'s "100 million builders" thesis (see [[Everyone Is a Builder]]). Profiled in depth in the sibling vault [[agentic-dev-envs/v0]]; this note covers the product-engineering lens.

## The Product-Engineering Angle

- Positioned as *"a design engineer tool"* — in the lineage of VS Code or Figma — not just a code generator. It embeds best practices so non-designers ship polished output, which Rauch frames as *democratizing* [[Craft as a Moat|craft]].
- **Mid-2025 inflection:** added persistent project context, multi-file generation, and direct Vercel deploy — repositioning from "component generator" to "full-stack app builder," expanding the audience beyond developers. This is [[Vercel]] pulling *left* in [[The Product Engineering Stack|the stack]] from ship into prototype/build (see [[Competitive Dynamics]]).
- Rauch's usage guidance is a [[Taste as the Bottleneck|taste]] lesson: feed it *references and inspiration* over prescriptive instructions, "because the AI might implement solutions better than you would have."
