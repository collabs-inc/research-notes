# ChatGPT Canvas

OpenAI's side-panel editor for writing and coding, where React/HTML code renders in a browser sandbox and Python executes in-browser. The OpenAI analog to [[Claude Artifacts]].

- **Launched** 3 Oct 2024 (Plus/Team first; free to all in early 2025), later folded into the default writing/coding flow.

## How HTML Is the Artifact

"React/HTML code is rendered in a sandbox environment" so you can preview the output; Python runs client-side via a Run button. Notably, Canvas can make external API requests — a wider sandbox than [[Claude Artifacts]], which [[Simon Willison]] flagged.

## The Distinction

Canvas is primarily a collaborative *editing surface* — highlight a section, version history, inline edits — rather than a host-and-share artifact platform. The render is a preview of the current code, not a persisted, shareable, hosted app. For OpenAI's host-and-embed story, see the [[OpenAI Apps SDK]].
